Cenușa de Oțel

Invața programare intr-un mod interactiv și distractiv!

  • Joaca acum

Despre proiect

Un joc in stil Hour Of Code in care te foloesti de blocuri de cod pentru a controla un robot

Tehnologia folosita

Pentru programarea jocului s-a folosit Godot, iar pentru ilustrarea texturilor s-a folosit Krita

Scopul proiectului

Prin acest proiect speram sa introducem conceptul de programare la cat mai multi elevi

Dispozitive compatibile

Jocul ruleaza pe orice dispozitiv, desktop, telefon, tableta, laptop, etc

Gazduirea site

Site-ul este gazduit folosit Github Pages, iar jocul propriu-zis se foloseste de Itch.io

Informatii aditionale
